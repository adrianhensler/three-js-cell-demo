About this demo

A fully interactive 3D human-cell scene built as a single self-contained HTML file — no build tools, no server, no external assets. All geometry, materials, and lighting are authored in code.

Hover and click organelles to inspect structure details and complete the exploration tracker for all six core components.

The prompt

Build a polished interactive 3D human-cell demo as a single-file index.html using Three.js from CDN only. No npm, no build tools, no external assets. Include hover labels, click interactions, objective/status tracking, atmospheric motion, and a completion state when all target organelles have been explored.
